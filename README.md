Nimma Guru is a modern Android application built using Kotlin and Firebase that helps users connect with mentors (Gurus), discover learning sessions, manage profiles, and engage with the community through appreciation posts and events.Features
Authentication
Phone number OTP authentication
User onboarding flow
Role-based user profiles
Guru Discovery
Search and discover Gurus
View mentor profiles
Explore expertise and ratings
